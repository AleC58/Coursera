es19 User Authentication with Passport and JSON Web Token

Installare passport-jwt e jsonwebtoken:

npm install passport-jwt jsonwebtoken --save

Creare nella root del progetto il file config.js, nel quale mettere info di configurazione.
In authenticate.js creare la JwtStrategy
